# 连接器的使用

本项目是一个 SSO 和 Apollo 的复合项目，支持单点登录、单点注销、环境变量推送等功能，为了在各个同组应用间共享状态，特提供连接器模块 node-sso-connect。

# node-sso-connect

node-sso-connect 模块可与 node-sso-server (本项目)通讯，实时获取到单点注销和环境变量更新等通知，该模块已发布到 npmjs，可以直接使用。

# Installation

$ npm install node-sso-connect --save

import nsc from 'node-sso-connect';

// 连接sso服务
nsc.connect({
	url: 'ws://localhost:3001',
	appid: 'appid',
	appsecret: 'appsecret',
	onConnect: () => {},
	onLogout: () => {},
	onPublish: () => {},
	reStep: 60,
});

# Attribute

参数 类型 必填 说明
url string 是 sso socket 地址（也就是本项目的 socket 地址）
appid string 是 应用账号（请联系管理员提供）
appsecret string 是 应用私钥（请联系管理员提供）
onConnect function 否 连接成功时触发
onLogout function 否 单点注销时触发
onPublish function 否 环境变量发布时触发
onError function 否 发生异常时触发
reStep Number 否 断线重连检测间隔（秒），默认 60

# 注意事项

  • 建议 node-sso-connect 全局唯一，只实例化一次
