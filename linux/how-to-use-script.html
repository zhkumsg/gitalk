# 常用命令的使用

对于命令行的接触，最开始是window下的cmd，比如查看电脑ip地址、进入某一个文件夹、下载npm模块等等

window   cmd

window  ipconfig

对于习惯了window桌面操作系统的用户来说，其实是比较少接触到命令行的，而在linux下，命令操作是常态。

linux命令不少，不过不用担心，常用的其实不多，而且有一些还是与window互通的，实在不知道具体命令，也可以使用百度。

下面我列出一下常用的linux操作命令

1、查看IP地址（相当于window下的ipconfig）

ifconfig

2、打开文件夹（同window的cd）

cd  ./xxx  //进入当前路径下的xxx文件夹

cd  ~   //进入用户根目录、同window下的用户根目录

cd  /    //进入根目录

cd  /usr/xxx   //进入根目录下的xxx文件夹

3、查看文件目录（同window的dir）

ls        //查看文件列表，常用

ls  -a    //查看文件列表，包括隐藏的

ls  -l     //查看文件列表，包括权限及软连接指向

4、查看当前路径

pwd

5、查看编辑文件

vim  xxx    //后续介绍

6、删除文件/文件夹

rm  -rf   xxx

7、下载文件

wget   -c     xxx.zip

8、设置权限（类似window下的以管理员身份运行）

chmod   -R   xxx   -777

9、安装yum模块（类似npm install）

yum   install    xxx

10、设置软连接（类似于window下的快捷方式或环境变量）

ln  -s   原路径    目标路径

11、查看系统版本

uname  -a    //查看操作内核信息

cat /etc/redhat-release   //查看发行信息

12、后台运行命令

nohup   xxx   &

13、查看任务

jobs

ps aux | less

14、退出终端

exit

15、重启服务器

reboot
