    # mysql的安装与使用

    对于网站或其他应用来说，数据是最重要，而数据既可以存在文本文件中，也可以存在内存中，为了方便管理，一般都会数据库来存储数据。其中mysql就相当流行一款关系型数据库，而且还支持在linux和window等操作系统是运行，这里我介绍如何在linux服务器上安装mysql。

    mysql

    # 1、选择安装方式

    不像window上一键安装mysql，在linux上安装会相对复杂一些，我们可以采用编译源码的形式，可以使用Yum一键安装，也可以使用APT安装，为了方便，这里使用Yum的形式安装。

    当然，在开始安装之前我们得检查一下是否已经安装了mysql。远程登陆服务器后，我们输入下面的命令检查一下

    whereis   mysql

    也可以使用

    which   mysql

    这两句命令可以检查出是否已安装mysql，如果已安装会显示出安装路径。

    # 2、下载rpm资源包

    直接使用yum是无法安装高级版的mysql的，因此我们需要先去官网下载合适的rpm资源包，官网链接是https://dev.mysql.com/downloads/repo/yum/

    选择合适的rpm

    然而下载最新的rpm包往往会出现安装不上的问题，所以在这里我推荐使用稍微旧一点的资源包，这里我推荐使用http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm

    我们可以把该资源下载到本地，使用wget下载

    wget   -c   http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm

    下载后保存的位置没有要求

    # 3、更新rpm资源包

    下载下来后，我们更新进本地rpm中

    rpm   -Uvh   mysql-community-release-el7-5.noarch.rpm 

    当然聪明的你会知道使用tap键自动补全文件名的。回车后等待一段时间，会提示更新成功success

    接着我们看一下更新rpm后我们可用的mysql版本是什么

    yum repolist enabled | grep "mysql.*-community.*"

    mysql可用版本

    # 4、安装mysql

    资源都已经准备好了，我们就可以安装mysql了。一般来说，我们需要安装mysql-server和mysql-client，不过因为我们是服务器，所以只安装server端就行了

    yum   -y   install   mysql-community-server

    注：安装社区版mysql服务端

    yum会自动安装相关依赖，等待一段时间后（大概十来分钟），安装成功。

    这时候可以用步骤一的命令检查是否安装成功。

    # 5、配置开机启动

    对于一个稳定的应用，当然希望服务一直启动着，万一断电后重启，也要尽快的把服务启动起来，所以我们把mysql服务配置成开启自启

    systemctl   enable   mysqld

    注意是mysqld，不是mysql，mysqld的是mysql的关键服务部分

    # 6、启动mysql服务

    安装成功后mysql服务不会自己启动的，我们需要去启动它

    systemctl   start   mysql

    回车后如果正常，是不会输入任何信息的，这时候我们可以通过status查看一下当前状态

    systemctl   status   mysql

    查看数据库状态

    # 7、创建数据库用户

    安装完数据库后我们需要创建管理员，使用mysqladmin命令即可，如希望创建用户为root，密码为111111（密码太简单数据库会提示升级密码的，不过可以忽略），那么命令如下

    mysqladmin     -u   root   password   111111

    创建完成后我们可以登录进去查看内容，登录命令是

    mysql   -u root -p

    回车后填写密码即可

    登录数据库

    # 8、查看数据库内容

    到这里就是简单的数据库操作了，比如我们可以查看一下有多少个数据库在里面

    show  databases;

    注：是databases而不是database，而且需要加分号

    show  databases;

    # 9、允许远程访问

    在一些情况下，该数据库还需要开放给外网使用（默认是本机使用），这时候还需要去配置一下权限，这里以不限制ip为例（下面代码均在本机登陆mysql后执行）

    grant all privileges on *.* to 'root'@'%' identified by '111111' with grant option;

    其中root是你的账号，111111是你的密码。

    修改后还需要刷新一下缓存才可以生效

    flush privileges;

    # 10、远程登陆

    远程登陆有很多种形式，比如我们在window上安装mysql-client登陆，也可以使用navicat登陆，这里就不再展开了
